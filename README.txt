To run the application, the following steps need to be executed first:
• Download Node on the machine from https://nodejs.org
• Run `npm create-react-app folder_name` in command prompt to create a react application folder
• Open the application folder in command prompt and run `npm i @tensorflow/tfjs @tensorflow-models/handpose react-webcam fingerpose` to download and install all the necessary dependencies and modules to run the application
• Open the application folder in command prompt and run `npm start` to compile the application and open it in the browser
